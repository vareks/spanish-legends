Concepto

La idea de esta web es transmitir cercanía, apetito y facilidad para consultar carta o llamar. Encaja muy bien una home con destacados de pizzas, calzones y postres.

El tono visual está pensado para reforzar una experiencia acogedora, italiana y accesible para público local y turista.
